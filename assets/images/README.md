# Images Folder Guide

This folder stores all website images.

Suggested structure:

- home: homepage banners and section images
- members: profile photos
- publications: journal covers or figure thumbnails
- gallery: activity and event photos
- logos: university and group logos
- icons: custom icon files

Naming suggestion:

- Use lowercase English names with hyphens, e.g. group-photo-2026.webp
- Prefer webp for photos; use svg/png for transparent graphics
